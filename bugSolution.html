The correct way to apply a conditional style is to use CSS and avoid the direct inclusion of less than/greater than symbols within the inline style. For example:

<style>
div {
  width: 100px;
}
div.lessThan100 {
  width: 50px;
}
</style>

<div class="lessThan100">This div's width is less than 100px.</div>

Alternatively, use JavaScript to dynamically set the style:

<div id="myDiv"></div>
<script>
  const div = document.getElementById('myDiv');
  let width = 50; // Dynamically set the width
  div.style.width = width + "px";
</script>
